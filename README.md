# Telegram Classic

Telegram Classic is a custom Telegram client built for iOS 5 and 6. It aims to recreate the experience of Telegram back in 2013, while adding modern touches.

### This project is in heavy alpha stages. The engine needs a lot of work to fix various networking issues as well as app hangs and crashes.


# How to build
This project is built with Xcode 4.6.3, using the iOS 6.1 SDK. Please note that the project will not compile for the iOS simulator at this time. All testing must be performed on real hardware, for the engine and various libraries are not compiled for i386, only armv7 and x86_64.
